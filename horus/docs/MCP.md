# MCP Architecture

MCP is a first-class tool fabric in Horus.

## Concepts
- Server registry loaded from `horus.yaml`.
- Per-agent MCP permissions.
- Tool schema discovery and caching.
- Health checks and dry-run simulation.
- Tool-call tracing into observability and memory.

## Core Integrations
filesystem, git, GitHub, terminal, browser, web search, databases, Docker, cloud, issue trackers, CI/CD, design tools, editors.
