# Memory System

Horus memory is layered and user-correctable.

## Layers
- Ephemeral context: active runtime state.
- Session memory: transcripts, commands, files touched.
- Project memory: architecture, tests, conventions, deployments.
- User memory: preferences and recurring context.
- Skill memory: reusable procedures.
- Semantic memory: vector-ready summaries.
- Full-text memory: SQLite FTS5.
- Event memory: structured tool/model/agent events.

## Operations
search, summarize, compress, delete, export, import, scoped access, optional encryption.
