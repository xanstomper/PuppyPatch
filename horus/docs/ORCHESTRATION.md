# Orchestration

Horus coordinates up to 60 simultaneous agents through explicit task contracts.

## Contract Fields
name, role, model, tools, workspace, permissions, memory access, budget, max runtime, objective, allowed file scope, exit criteria, reporting format.

## Runtime Features
Isolated workspaces, dependency DAGs, shared blackboard, branch-per-agent option, review gates, human approvals, budget enforcement, runaway protection, event streaming.
